# chengyue_web
通过vue2.0搭建的门户网站，为实现网站的轮播，文章展示，广告展示等功能。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
